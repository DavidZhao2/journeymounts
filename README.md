# Cobblemon Journey Mounts

## Overview

Cobblemon Journey Mounts (CJM) is a Fabric-based Minecraft mod that enhances the Cobblemon experience by introducing fully rideable Pokémon with advanced movement mechanics and customizable riding statistics.

The project focuses on performance, multiplayer stability, and extensible architecture. It expands traversal across land, air, and water while maintaining compatibility with the Cobblemon ecosystem.

This mod has surpassed 2 million downloads across releases.

---

## Key Features

**Advanced Riding Systems**
- Land, water, and air traversal mechanics
- Custom acceleration, speed, jump, skill, and stamina attributes
- Balanced ride stat scaling

**Multiplayer Compatibility**
- Tested across dedicated servers
- Synchronized riding behavior
- Stable entity handling

**Multi-Seat Support**
- Expanded seating capacity for supported Pokémon
- Optimized passenger positioning

**Custom Structures (WIP)**
- Stables
- Markets
- Race tracks
- Thematic world integrations

**Performance Optimization**
- Clean architecture
- Modular design
- Gradle-based build system

## Installation

1. Install Fabric Loader.
2. Install Cobblemon.
3. Place the CJM `.jar` file into the `mods` folder.
4. Launch Minecraft.
